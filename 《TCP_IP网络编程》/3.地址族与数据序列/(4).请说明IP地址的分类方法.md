IP地址的分类方法主要依据其第一字节（即第一个十进制数）的取值范围进行划分，共分为A、B、C、D、E五类：

- A类地址：第一字节范围为 1~126，适用于大型网络，网络号占第一个字节，主机号占后三个字节。

- B类地址：第一字节范围为 128~191，适用于中型网络，网络号占前两个字节，主机号占后两个字节。

- C类地址：第一字节范围为 192~223，适用于小型网络，网络号占前三个字节，主机号占最后一个字节。

- D类地址：第一字节范围为 224~239，用于多播（组播）通信，不分配给单个主机。

- E类地址：第一字节范围为 240~255，保留用于实验和研究，不用于常规网络通信。

因此，判断一个IP地址属于哪一类，只需查看其第一个十进制数落在哪个区间：

- 若第一个数在 1~126 → A类；

- 若第一个数在 128~191 → B类；

- 若第一个数在 192~223 → C类；

- 若第一个数在 224~239 → D类；

- 若第一个数在 240~255 → E类。

注意：0和127有特殊含义，0表示当前网络，127是回环地址（如127.0.0.1），不属于标准A类地址范畴。

综上，IP地址分类依据其首字节范围，并据此可快速判断其类别。

答案： 

IP地址根据首字节范围分为A、B、C、D、E五类：A类（1~126）、B类（128~191）、C类（192~223）、D类（224~239）、E类（240~255）。判断某IP地址类别，只需看其第一个十进制数落在哪个区间即可。
